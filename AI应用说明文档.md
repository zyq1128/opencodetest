### 1.3 核心算法说明

#### 1.3.1 Isolation Forest 异常检测

**算法原理：**
Isolation Forest（孤立森林）是一种基于树的无监督异常检测算法。其核心思想是：异常点由于与正常数据分布不同，更容易被随机分割"孤立"出来。

**实现细节：**
1. **基线预训练**：使用600条合成数据进行预训练
   - 正态分布模拟正常工况参数
   - 注入30条异常数据（压力突升、电流异常、重量离散）
   
2. **特征工程**：6维输入特征
   - 成型压力、成型节拍（核心工艺参数）
   - 上冲/中模伺服电流（设备磨损指标）
   - 称重标准差、称重均值（质量稳定性）

3. **输出解释**：
   - 异常标签：-1=异常，1=正常
   - 异常得分：绝对值越大越异常（通常>0.5触发报警）
   - 异常率：近期异常占比

#### 1.3.2 Random Forest 良率预测

**算法原理：**
Random Forest（随机森林）是一种集成学习算法，通过构建多棵决策树并投票/平均来提高预测准确性和鲁棒性。

**实现细节：**
1. **训练数据**：1000条合成历史数据
   - 基于工艺参数范围生成合理值
   - 根据工艺逻辑标注良率标签
   
2. **特征工程**：11维输入特征
   - 填充高度、压制高度（装粉相关）
   - 成型压力、保压时间、压制速度、脱模速度（压制工艺）
   - 成型节拍（效率指标）
   - 上冲/中模伺服电流（设备状态）
   - 称重均值（质量指标）

3. **模型参数**：
   - n_estimators=100（100棵树）
   - max_depth=8（最大深度限制过拟合）
   - 特征重要性计算提供可解释性

4. **输出解释**：
   - 预测良率：0~1之间的概率值
   - 质量等级：优秀(≥90%) / 良好(≥80%) / 警告(≥70%) / 差(<70%)
   - 特征重要性：条形图展示各工艺参数对良率的影响程度

#### 1.3.3 健康评分算法

**算法原理：**
结合规则引擎和统计方法，从多个维度综合评估设备健康状态。采用滑动窗口（最近60条数据）实时计算。

**实现细节：**
1. **评分维度**（4项，每项0-100分）：
   
   a) **压力稳定性评分**
   - 计算近60条成型压力的变异系数（CV = 标准差/均值）
   - 评分公式：100 - CV × 600
   - CV越小（压力越稳定），得分越高
   
   b) **节拍稳定性评分**
   - 计算近60条成型节拍的标准差
   - 评分公式：100 - 标准差 × 500
   - 节拍越均匀，得分越高
   
   c) **报警评分**
   - 统计近期报警频率
   - 无报警满分，报警越频繁扣分越多
   
   d) **伺服电流评分**
   - 上冲/中模伺服电流与正常基准的偏差
   - 基准值：上冲8.0A，中模6.0A
   - 偏差越大得分越低（提示模具磨损或润滑不良）

2. **综合评分计算**：
   - 默认：4项评分简单平均
   - 支持加权配置（可根据设备类型调整权重）

3. **老化预警机制**：
   - 检测压力持续上升趋势（线性回归斜率持续为正值）
   - 触发条件：连续多个窗口斜率>阈值
   - 提示信息：「老化预警」+ 斜率值
   - 含义：设备可能因磨损导致压力漂移，建议维护

4. **输出解释**：
   - 综合评分：0-100分
   - 健康等级：
     * 优秀：90-100分（绿色）
     * 良好：75-89分（蓝色）
     * 警告：55-74分（黄色）
     * 危险：0-54分（红色）
   - 分项评分：4个维度各自的得分条形图
   - 老化预警：布尔标志+斜率值

#### 1.3.4 趋势预测算法

**算法原理：**
使用线性回归（Linear Regression）对时间序列数据进行建模，预测未来趋势。支持多通道同时预测。

**实现细节：**
1. **支持通道**（4个关键工艺参数）：
   - 成型压力（cxyl）：单位MPa，默认基准175.0
   - 成型节拍（cxjp）：单位秒，默认基准3.5
   - 上冲伺服电流（scsfdl）：单位A，默认基准8.0
   - 中模伺服电流（zmsfdl）：单位A，默认基准6.0

2. **模型参数**：
   - 历史窗口：最近20条数据
   - 预测步长：10步（约未来10秒）
   - 算法：scikit-learn LinearRegression
   - 输入：时间步序号（0,1,2...n）
   - 输出：预测值序列

3. **趋势标签算法**：
   根据回归斜率判断趋势：
   - 快速上升⚠️：斜率 > rise_fast阈值
   - 缓慢上升：rise_slow < 斜率 ≤ rise_fast
   - 稳定：-rise_slow ≤ 斜率 ≤ rise_slow
   - 缓慢下降：-rise_fast ≤ 斜率 < -rise_slow
   - 快速下降⚠️：斜率 < -rise_fast
   
   各通道阈值配置：
   - 成型压力：rise_fast=1.0, rise_slow=0.2 (MPa/步)
   - 成型节拍：rise_fast=0.5, rise_slow=0.1 (秒/步)
   - 上冲电流：rise_fast=0.3, rise_slow=0.05 (A/步)
   - 中模电流：rise_fast=0.3, rise_slow=0.05 (A/步)

4. **特殊功能（仅成型压力Tab）**：
   - 黄色虚线：AI预测曲线，叠加在历史数据上
   - 摘要行显示：
     * 当前压力：最新实测值
     * 趋势斜率：线性回归斜率（MPa/采样）
     * 预测10s后峰值：未来10步预测最大值
     * 趋势判断：上述5种标签之一

5. **输出解释**：
   - 预测值序列：10个未来预测值
   - 回归斜率：浮点数，正值上升、负值下降
   - 趋势标签：5种状态之一，带⚠️表示需要关注
   - R²得分：可选，表示模型拟合度（0-1之间）

## 三、AI模块技术规格

### 3.1 异常检测模块（anomaly.py）

#### 算法详情
- **算法类型**：Isolation Forest（孤立森林）
- **学习方式**：无监督学习
- **适用场景**：无需标注数据，自动发现异常模式

#### 数学原理
孤立森林通过随机选择特征和分割值来构建多棵树。异常点因为与正常数据分布不同，通常在树的上层就被孤立出来（路径长度短）。

异常分数计算公式：
```
s(x, n) = 2 ^ (-E(h(x)) / c(n))
```
其中：
- E(h(x))：样本x在多棵树中的平均路径长度
- c(n)：给定样本数n时的平均路径长度
- s接近1表示异常，接近0.5表示正常

#### 实现参数
```python
IsolationForest(
    n_estimators=100,      # 树的数量
    contamination=0.05,    # 异常比例假设（5%）
    random_state=42,       # 随机种子（保证可复现）
    max_samples='auto'     # 每棵树采样数
)
```

#### 基线预训练机制
为避免冷启动问题，模型使用合成数据进行预训练：

1. **正常数据生成**（570条）：
   - 成型压力：N(175.0, 5.0)
   - 成型节拍：N(3.5, 0.2)
   - 上冲电流：N(8.0, 0.5)
   - 中模电流：N(6.0, 0.4)
   - 称重标准差：|N(0.3, 0.1)|
   - 称重均值：N(45.5, 0.8)

2. **异常数据注入**（30条）：
   - 压力突升：+25~40 MPa
   - 电流异常：+3~6 A
   - 重量离散：+1~3 g标准差

#### 输出字段说明
```json
{
  "is_anomaly": true,           // 是否异常
  "anomaly_label": -1,          // -1=异常, 1=正常
  "anomaly_score": -0.6234,     // 异常分数（负值越大越异常）
  "anomaly_count": 5,           // 近期异常数量
  "anomaly_rate": 0.05,         // 异常率（5%）
  "sample_size": 100,           // 样本总数
```

### 3.2 良率预测模块（quality.py）

#### 算法详情
- **算法类型**：Random Forest Regressor（随机森林回归）
- **学习方式**：监督学习
- **适用场景**：基于工艺参数预测产品质量

#### 数学原理
随机森林通过构建多棵决策树并取平均来进行预测：

```
ŷ = (1/B) * Σb=1^B fb(x)
```
其中：
- B：树的数量（n_estimators）
- fb：第b棵决策树
- x：输入特征向量

特征重要性计算（Gini重要性）：
```
Importance(xi) = Σt∈T (p(t) * Δi(xi, t))
```
其中p(t)是节点t的样本占比，Δi是Gini不纯度减少量。

#### 实现参数
```python
RandomForestRegressor(
    n_estimators=100,      # 树的数量
    max_depth=8,           # 最大深度（防止过拟合）
```

### 3.3 健康评分模块（health.py）

#### 算法详情
- **算法类型**：规则引擎 + 统计方法（混合算法）
- **学习方式**：基于规则的实时计算
- **适用场景**：多维度设备状态综合评估

#### 评分维度详解

**1. 压力稳定性评分（权重25%）**
```python
def calculate_pressure_stability(pressures: list) -> float:
    mean = np.mean(pressures)
    std = np.std(pressures)
    cv = std / mean if mean > 0 else 0  # 变异系数
    
    # 评分公式：100 - CV × 600
    score = max(0.0, 100.0 - cv * 600)
    return score
```
- 原理：变异系数（CV）衡量相对离散程度，不受量纲影响
- 阈值：CV=0.1时得分40分，CV=0.17时得分0分

**2. 节拍稳定性评分（权重25%）**
```python
def calculate_cycle_stability(cycle_times: list) -> float:
    std = np.std(cycle_times)
    # 评分公式：100 - 标准差 × 500
    score = max(0.0, 100.0 - std * 500)
    return score
```
- 原理：节拍标准差直接反映生产节奏稳定性
- 阈值：标准差0.2s时得分0分

**3. 报警评分（权重25%）**
```python
def calculate_alarm_score(alarm_history: list, window: int = 60) -> float:
    alarm_count = sum(alarm_history[-window:])
    alarm_rate = alarm_count / window
    
    # 评分公式：100 × (1 - 报警率)
    score = 100 * (1 - alarm_rate)
    return max(0.0, score)
```
- 原理：近期报警频率越低，得分越高
- 特点：简单直观的规则，无报警时满分

**4. 伺服电流评分（权重25%）**
```python
def calculate_servo_current_score(
    sc_currents: list,  # 上冲电流
    zm_currents: list   # 中模电流
) -> float:
    # 正常基准值
    SC_NORMAL = 8.0   # 上冲
    ZM_NORMAL = 6.0   # 中模
    
    # 计算平均偏差
    sc_mean = np.mean(sc_currents)
    zm_mean = np.mean(zm_currents)
    
    sc_dev = max(0, sc_mean - SC_NORMAL)
    zm_dev = max(0, zm_mean - ZM_NORMAL)
    
    # 评分公式
    sc_score = max(0, 100 - sc_dev * 12)
    zm_score = max(0, 100 - zm_dev * 12)
    
    return (sc_score + zm_score) / 2
```
- 原理：电流偏离正常基准越大，提示模具磨损或润滑不良越严重
- 特点：仅对正向偏离敏感（电流偏高更危险）

#### 老化预警算法

```python
def detect_aging_warning(pressure_history: list, window: int = 60) -> dict:
    """
    检测压力持续上升趋势，预警设备老化
    """
    if len(pressure_history) < window:
        return {"warning": False, "slope": 0}
    
    # 取最近window条数据
    recent = pressure_history[-window:]
    
    # 线性回归拟合
    x = np.arange(len(recent))
    slope, intercept = np.polyfit(x, recent, 1)
    
    # 判断条件：斜率持续为正且超过阈值
    threshold = 0.05  # MPa/步
    warning = slope > threshold
    
    return {
        "warning": warning,
        "slope": round(slope, 3),
        "message": "设备老化预警（压力上升趋势）" if warning else "正常"
    }
```

**触发条件**：
1. 线性回归斜率持续为正值（压力上升）
2. 斜率超过阈值（默认0.05 MPa/步）
3. 连续多个窗口满足条件

**物理意义**：
- 设备长期运行后，模具磨损导致压力漂移
- 为保持相同压制效果，系统需要更高压力
- 提示需要维护：更换模具、润滑保养等
